# GuiaPractica01-AgendaTelefonica-CSharp

Sistema de Agenda Telefónica en C# con Programación Orientada a Objetos.

## Estructura del Proyecto

```
GuiaPractica01-AgendaTelefonica-CSharp/
├── codigo/              # Código fuente (.cs)
│   ├── Program.cs
│   ├── Agenda.cs
│   ├── Contacto.cs
│   └── Estadisticas.cs
├── base/                # Archivos de configuración y documentación
│   ├── Agenda.csproj
│   ├── README.md
│   └── ANALISIS.md
├── compilado/           # Archivos compilados (.exe, .dll)
│   ├── Agenda.exe
│   └── Agenda.dll
└── .gitignore
```

## Compilar y Ejecutar

```bash
dotnet build
dotnet run
```

## Funcionalidades

1. Agregar contacto
2. Listar contactos
3. Buscar contacto por nombre
4. Registrar llamadas por mes
5. Ver estadísticas de llamadas
6. Ver top contactos por cantidad de llamadas

## Requisitos

- .NET 8.0 SDK
